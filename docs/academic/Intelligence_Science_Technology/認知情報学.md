# 認知情報学研究室

## 研究室简介

認知情報学研究室致力于研究人类感知和认知的计算模型，探索人类智能的基本原理，并将这些原理应用于开发新一代的智能系统。

## 主要研究方向

- 视觉感知计算模型
- 多感官信息整合
- 认知计算理论
- 人工智能系统设计

## 研究项目

1. 人类视觉系统的计算模型
2. 跨模态感知整合机制
3. 认知系统的工程实现
4. 智能感知系统开发
