# 心理情報学研究室

## 研究室简介

心理情報学研究室专注于研究人类认知过程的计算理论，将心理学实验与信息科学方法相结合，探索人类认知的本质。

## 主要研究方向

- 视觉注意机制研究
- 认知过程的计算建模
- 人类决策行为分析
- 心理物理学实验方法

## 研究项目

1. 视觉搜索的认知机制
2. 人类决策的计算模型
3. 注意力分配的神经基础
4. 认知负荷的量化分析
