# 人間拡張情報研究室

## 研究室简介

人間拡張情報研究室致力于研究如何通过信息技术增强人类的认知和交互能力。我们探索创新的人机交互方式，开发能够扩展人类能力的智能系统。

## 主要研究方向

- 人机交互技术
- 认知增强系统
- 虚拟现实与增强现实
- 可穿戴计算设备
- 人工智能辅助技术

## 研究项目

1. 智能交互界面开发
2. 认知辅助系统
3. AR/VR应用研究
4. 可穿戴设备开发
5. 人工智能交互设计
