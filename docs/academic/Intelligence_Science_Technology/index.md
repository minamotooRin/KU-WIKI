# 知能情報学专攻

## 专攻简介

知能情報学专攻致力于构建和阐明智能的本质，实现灵活的类人信息处理。在高度信息化社会中，具备灵活的、类人的信息处理能力变得越来越重要。人类和动物的有机系统所具有的信息处理能力是无与伦比的，这种能力是通过在漫长的进化过程中对环境的结构性和功能性适应而发展起来的。知能情報学是一个多学科领域，旨在阐明生物特别是人类信息处理的机制，从而发展更高级的信息处理方法。

## 研究方向

- 机器学习与深度学习
- 计算机视觉
- 自然语言处理
- 知识工程
- 智能机器人技术

## 下属研究室

- [计算机视觉组](计算机视觉组.md)：研究视觉信息处理
- [自然语言处理组](自然语言处理组.md)：致力于语言智能研究
- [知识工程组](知识工程组.md)：研究知识表示与推理
