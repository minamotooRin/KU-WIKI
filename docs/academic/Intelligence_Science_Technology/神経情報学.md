# 神経情報学研究室

## 研究室简介

神经信息学研究室致力于运用信息科学方法研究大脑功能，探索人类智能的本质。我们主要研究大脑的信息处理机制，开发基于脑科学的智能系统。

## 主要研究方向

- 脑活动数据的机器学习分析
- 视觉信息处理的神经机制研究
- 认知神经科学计算模型
- 脑机接口技术开发

## 研究项目

1. 基于深度学习的脑活动解码
2. 视觉感知的计算理论研究
3. 神经网络动力学建模
4. 脑信息处理的数学模型构建
