# 数学物理组

## 研究方向

- 量子力学的数学理论
- 场论与粒子物理
- 统计物理数学基础
- 相对论的数学描述
- 数学物理中的几何方法

## 研究项目

- 量子场论的数学结构研究
- 引力理论的数学基础
- 可积系统的数学分析
- 统计力学的严格推导
- 物理系统的数学建模
