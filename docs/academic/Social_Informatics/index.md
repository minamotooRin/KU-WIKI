# 社会情报学专攻

## 专攻简介

社会情报学专攻研究信息技术与社会的交互关系。本专攻涵盖信息系统设计、信息安全、数字图书馆、医疗信息等多个领域。

## 研究方向

- 分布式信息系统
- 信息安全
- 医疗信息学
- 数字图书馆
- 信息社会学
