# 医疗情报学研究室 / Medical Informatics Lab

[研究室主页](https://medinfo.kuhp.kyoto-u.ac.jp/)（日文）

教授 / PI: 黑田知宏 (Kuroda Tomohiro)

短介绍 / Summary

本组专注于医疗现场数据与临床流程的理解，主要研究方向包括：手术室状态感知（surgical OR understanding）、电子病历(electronic health records)的数据建模与应用，以及医疗图像与时序数据的融合分析。适合对医疗现场问题、传感数据和机器学习系统工程感兴趣的同学。

研究室规模与国际生情况

```tip
通常每年招收少量学生（约2–3名），外国留学生的名额有限；博士阶段接受国际学生的比例较高。
```

入学 / 毕业要求

- 硕士：参与研究并完成论文与学位要求（依校方/院系规定）。
- 博士：在国际/国内学术会议或期刊上发表论文（一般至少 2 篇学术发表为参考目标）。

日常与例行活动

- 每周一 09:00–12:00：研究会（进展汇报、论文讨论）
- 每周五 09:00–12:00：学习会 / 实验分享

设备与资源

- 服务器：32 cores, 128 GB RAM，若干 GPU（历史使用 Titan 系列）与虚拟机环境
- 常用软件栈：Python、PyTorch、常见医疗数据处理工具

如何加入（建议流程）

1. 联系教授或相关在读学生，说明你的研究兴趣与背景。
2. 提交简历（英文/日文均可）、成绩单与研究计划草案。
3. 视具体情况安排面试（线上/线下）。

实用提示

- 如果你是国际学生，提前准备好奖学金/助研信息并咨询导师助学金可能性。
- 若不熟悉医疗数据，请先熟悉基本的医学术语与隐私法规（HIPAA/日本相关规范）。
