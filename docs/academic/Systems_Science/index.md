# 系统科学专攻

## 专攻简介

系统科学专攻专注于研究复杂系统的建模、分析和控制。本专攻整合了控制理论、系统工程和计算机科学等领域的知识，旨在解决现代社会中的复杂系统问题。

## 研究方向

- 系统理论
- 人工生命
- 复杂网络
- 控制工程
- 机器人技术

## 下属研究室

- [系统理论组](系统理论组.md)：研究系统建模与分析
- [人工生命组](人工生命组.md)：研究生命系统仿真
- [复杂网络组](复杂网络组.md)：研究网络系统动力学

## 学习环境

本专攻配备有系统仿真实验室、机器人实验平台和高性能计算设施，为学生提供完善的研究环境。

## 就业方向

- 自动化企业
- 研究机构
- 系统集成公司
- 咨询公司
- 高科技企业
