# 常见问题 (FAQ)

## 入学相关

??? question "京大好考吗？"
    取决于专攻和研究室。建议提前联系意向导师，了解研究室的招生情况和要求。

??? question "如何联系导师？"
    1. 在学校官网找到导师邮箱
    2. 发送简洁的自我介绍邮件，附上 CV 和研究计划
    3. 等待回复，通常需要 1-2 周

??? question "需要日语能力吗？"
    - 英语项目：不强制要求，但日常生活需要基础日语
    - 日语项目：通常需要 N2 以上
    - 建议至少掌握 N3 水平以便日常生活

## 签证与手续

??? question "COE 是什么？"
    在留资格认定证明书 (Certificate of Eligibility)。入学前由学校代为申请，获得后在中国申请签证。

??? question "到日本后要做什么？"
    1. **14天内**：到市役所办理住民登记
    2. **同时**：申请国民健康保险
    3. **尽快**：开设银行账户、办理手机

??? question "可以打工吗？"
    可以，但需要办理「资格外活动许可」。限制：

    - 每周最多 28 小时（假期可延长）
    - 不能从事风俗业

## 生活相关

??? question "怎么加入微信群？"
    关注 KU-WIKI 仓库 Issue 或联系已入学的学长学姐获取群二维码。

??? question "京大附近有什么中餐馆？"
    **百万遍周边**:

    - 顺和
    - 状元楼
    - 柏三叶
    - 妈妈家牛肉饭

    **出町柳周边**:

    - 大众食堂
    - 各种拉面店

??? question "住宿怎么找？"
    - **校内宿舍**：通过学校申请，竞争激烈
    - **校外租房**：通过不动产中介，需要保证人或保证公司
    - 详见 [住宿指南](../housing.md)

??? question "怎么办理国民健康保险？"
    1. 到住所所在区的市役所
    2. 携带在留卡和护照
    3. 填写申请表
    4. 保险费根据收入计算，留学生通常可申请减免

## 学业相关

??? question "学分怎么算？"
    各专攻要求不同，详见各学院教务网站。一般硕士需要 30 学分左右，博士需要完成研究指导和论文。

??? question "可以跨专攻选课吗？"
    大部分情况可以，但需要确认是否计入毕业学分。建议咨询教务。
