## 京大留学生日历（KU International Students Calendar）

这是我们为京都大学留学生维护的共享日历，包含校园活动、签证手续截止日期、节日假期、文化活动等内容。  
本日历由留学生社区协作维护，任何更新都会自动同步到订阅者的日历中。

---

## 📅 订阅方式

你可以直接使用下方链接在任意设备上订阅：

- **iCloud / macOS / iPhone 用户：**  
  点击下方链接后选择「订阅」即可：  
  👉 [订阅日历](webcal://p53-caldav.icloud.com/published/2/MjAwMDk1NTU0OTEyMDAwOaGolMLJNhfiOpqexfmIractK4PHSNYOl5T-EVLwZQN11FTfW8vEKORjPXQ50OiEnZ9sbjwte0Vd9kJtUdcgaJo)

- **Google Calendar 用户：**  
  1. 打开 [Google Calendar](https://calendar.google.com/)  
  2. 左侧点击 “其他日历” → “通过网址添加”  
  3. 粘贴以下地址：  
     ```
     webcal://p53-caldav.icloud.com/published/2/MjAwMDk1NTU0OTEyMDAwOaGolMLJNhfiOpqexfmIractK4PHSNYOl5T-EVLwZQN11FTfW8vEKORjPXQ50OiEnZ9sbjwte0Vd9kJtUdcgaJo
     ```

- **Outlook / Thunderbird 等客户端：**  
  支持 `.ics` / `webcal://` 订阅的客户端都可以直接使用上面的地址。

## 🌏 时区说明
- 本日历的默认时区为 **Asia/Tokyo (日本时间)**  
- 若在海外查看，请确认你的日历应用自动调整为本地时区。

---

## 📖 更新说明
- 任何人都可以通过 GitHub 提交活动建议或修正。  
- 新增或修改后，所有订阅者会自动同步更新（一般延迟约 30 分钟至 3 小时，取决于各应用）。
