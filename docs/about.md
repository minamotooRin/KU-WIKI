# 关于本项目

## 项目目标

**KU-WIKI** 是一个为在京都大学读研、将要入学或即将到达的中国院生而创建的实用中文信息库。我们的使命是：

- 整理入学、签证、生活的关键步骤与模板，减少信息焦虑。
- 汇聚前辈与同学的真实经验，让每一位新生都能快速适应京都大学生活与科研。
- 为京大中文学生群体提供一个开放、温暖的知识共享平台。

## 项目信息

- **创建时间**：2025 年
- **维护者**：京都大学信息学研究科的同学与志愿者
- **源代码**：基于 [MkDocs](https://www.mkdocs.org) 与 [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) 主题
- **开源协议**：[MIT License](https://opensource.org/licenses/MIT)（详见 LICENSE 文件）

## 主要内容

- 📘 快速指南与实用工具
- 🛂 签证与在留资格指南
- 🏠 住宿信息与找房建议
- 🏥 医疗与健康保险指南
- 💰 奖学金、经费与金融服务
- 🎓 科研方向与导师寻找
- 🍜 校园生活与兼职机会
- 🔗 精选友情链接与外部资源

## 免责声明

- 本项目由学生维护，**非京都大学官方网站**，所有信息均尽力确保准确性，但不承担任何法律责任。
- 如涉及官方政策、法律程序等，请以京都大学官网、日本政府官网或相关机构发布的信息为准。
- 本项目内容定期更新，但可能存在遗漏或过时信息，欢迎指正与补充。

## 更新频率

- 核心指南（签证、住宿、医疗）：根据政策变化或同学反馈即时更新。
- 新闻与事件：重大消息（如诺贝尔奖、学校公告）同步发布。
- 一般内容维护：根据贡献者贡献频率与社区反馈更新。
