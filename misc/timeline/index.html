    入学时间表 / Onboarding Timeline

    本页为在京都大学的中文新生提供一份按时间轴划分的入学准备与适应指南。根据你的情况选择相应的时间段并按步骤完成任务。

    📅 入学前 3-6 个月

    必做事项

    • [ ] 确认入学资格与录取通知：检查邮件、学校官网、研究科信息。
    • [ ] 准备签证所需文件：获取 COE、体检证明、资金证明等。
    • [ ] 联系未来导师或研究室：初步了解研究方向与期待。
    • [ ] 注册宿舍申请（若学校提供）：提前了解校内宿舍情况与申请截止。
    • [ ] 确认到达日期与安排：预订机票、安排接机或自行前往。
    • [ ] 学习基础日语：最少掌握 N4 或生活基础用语。

    重要链接

    • 签证与在留资格指南 — COE 申请、在留资格认定、签证办理流程。
    • 住宿指南 — 校内宿舍申请、校外租房平台与合同要点。
    • 京都大学官方入学页面：Admission at Kyoto University

    📅 入学前 1 个月

    准备工作

    • [ ] 最终确认来日期与航班信息。
    • [ ] 准备行李与必需品：
    • 重要文件（护照、签证、体检证、成绩单原件）。
    • 常用药物与医疗用品。
    • 日本的电子设备适配器。
    • 足够的现金（日元或美元，用于抵达后提取 ATM）。
    • [ ] 通知导师与研究室：确认到达时间、第一天报到地点与时间。
    • [ ] 提前订好第一晚住宿：宿舍、酒店或朋友家。
    • [ ] 购买短期手机卡或 SIM：便于抵达后联系。
    • [ ] 了解机场到京都的交通：机场快线、巴士或出租车。

    推荐工具

    • Google Maps 离线地图 — 下载京都及周边离线地图。
    • Suica/ICOCA 卡 — 了解日本交通卡种类（可在机场购买）。
    • Docomo/Softbank 短期 SIM 卡 — 机场售卖，通话 + 数据。

    📅 到达当周（第 1-7 天）

    第 1-2 天：抵达与入住

    • [ ] 到达机场后：
    • 购买 ICOCA 交通卡（可充值）。
    • 购买短期 SIM 卡或租赁 WiFi（如未提前购买）。
    • 乘坐机场快线或巴士前往京都。
    • [ ] 入住宿舍或酒店。
    • [ ] 换取日元（机场或便利店 ATM）。
    • [ ] 休息与调整时差。

    第 3-7 天：重要手续（须在 7 日内完成）

    住民登记（必须，7 日内）

    • [ ] 前往当地市役所（区役所）办理住民登记。
    • [ ] 所需文件：护照、在留卡、出国记录。
    • [ ] 获得住民票（后续所有手续的基础）。

    国民健康保険加入（必须，7 日内）

    • [ ] 地点：市役所健康保険部门或研究科国际学生办公室。
    • [ ] 所需文件：住民票、护照、在留卡。
    • [ ] 获得保険证（必须保管妥当）。

    其他重要事项

    • [ ] 开设日本银行账户：
    • 优先选项：日本邮政银行（要求较低，接受国际学生）。
    • 备选：メガバンク（三菱/三井/三和等，需多人推荐）。
    • 所需：护照、在留卡、住民票、学生证。
    • [ ] 购买手机卡或办理合约（长期使用）：
    • 主流运营商：Docomo、Softbank、au 等。
    • 所需：护照、在留卡、住民票、银行卡。
    • [ ] 前往研究科/学院报到：
    • 获取学生证、宿舍钥匙（若申请了宿舍）、账号与密码。
    • 了解办公时间、紧急联系方式、例会时间。

    购物与适应

    • [ ] 去便利店（FamilyMart、Lawson 等）购买日常用品。
    • [ ] 逛超市了解食品价格与品类。
    • [ ] 探索研究室周边与学校食堂。

    相关资源页面

    • 快速指南与实用信息 — 紧急联系方式、医疗就诊、常用技巧。
    • 健康与医疗 — 国民健康保険、医院与诊所位置、就诊流程。
    • 住宿指南 — 宿舍登记、校外租房注意。

    📅 第一个月（第 8-30 天）

    学业与研究准备

    • [ ] 参加学院的 orientation 与新生说明会。
    • [ ] 确认学期日程、课程注册截止、论文提交时间表。
    • [ ] 第一次与导师面谈：讨论研究计划、周报告时间、预期目标。
    • [ ] 加入研究室的课题讨论会或论文读书会。
    • [ ] 了解校内图书馆、计算机教室、研究设施的使用规则。

    生活适应

    • [ ] 办理垃圾分类与清洁用品：购置垃圾袋、清洁剂、垃圾桶。
    • [ ] 购买或租赁家具与家电（若未提供）。
    • [ ] 学习垃圾分类规则：京都的分类较细致，请查阅市政府指南。
    • [ ] 报名学校的日语课程（如需要）。
    • [ ] 参加学院/校级新生活动与 welcome party。

    社交与网络

    • [ ] 加入研究室/学院的 LINE 群、微信群或 Slack 频道。
    • [ ] 认识同届同学与前辈。
    • [ ] 了解学院食堂、学生宿舍活动、体育社团等。
    • [ ] 可选：参加中国学生会或留学生会活动。

    文件与行政

    • [ ] 注册学号与系统账号（邮箱、VPN、图书馆系统等）。
    • [ ] 了解学费支付方式与截止日期。
    • [ ] 申请所需的奖学金或经费（如尚未申请）。
    • [ ] 更新护照与在留卡信息（如有变化）。

    • 健康与医疗 — 牙医预约、眼镜店、体检安排。
    • 快速指南 — 校园生活、兼职机会、校内食堂等生活技巧。

    📅 第二个月至学期末

    学业稳定期

    • [ ] 进入正常研究生活：参加课程、论文指导、课题研究。
    • [ ] 建立每周例会与沟通习惯。
    • [ ] 如需发表或投稿，与导师讨论时间表。

    财务与福利

    • [ ] 申请或续期奖学金（MEXT、学内、学外等）。
    • [ ] 了解兼职规则与申报流程（每周工时上限、税务报告等）。
    • [ ] 注册税务编号（必要时）。

    长期规划

    • [ ] 制定学年研究计划与论文时间表。
    • [ ] 探索校内外的研究机会、交流项目或联合研究。
    • [ ] 了解毕业后的升学或就业路径。

    参考资源

    📞 应急与求助

    情况 联系方式 备注
    医疗紧急 119 日语或英语说明地点与症状
    警察/犯罪 110 说明身份与事件地点
    学籍/手续问题 研究科国际学生办公室 通常在学院楼内，工作时间 8:30-17:00
    宿舍紧急 宿舍管理员 / 值班室 钥匙遗失、漏水等

    经济困难 学生支援课 / 奖学金办公室 询问临时援助与贷款

    📝 常见问题（FAQ）

    Q：到达后多久可以开始研究？
    A：通常在入学 1-2 周后，完成行政手续与 orientation 后即可正式开始。期间可与导师沟通，了解研究室文化与期望。

    Q：如果住民登记超过 7 天会怎样？
    A：虽然法律上应在 7 天内完成，但大多数情况下只要尽快办理即可。建议抵达后的前两天优先完成。

    Q：可以在宿舍办理住民登记吗？
    A：通常可以，使用宿舍地址即可。只需携带护照、在留卡、出国记录。

    Q：研究室的例会一般多久开一次？
    A：因研究室而异，通常为每周一次（1-2 小时）或每两周一次。第一天与导师沟通时应明确询问。
